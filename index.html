추가 버튼 기능 설명

버튼 기능
x^y 거듭제곱 (Power)
x를 밑, y를 지수로 계산합니다.
x 입력 → x^y 클릭 → y 입력 → = 클릭 예) 2 x^y 10 = → 1024  |  3 x^y 3 = → 27
log 상용로그 (log₁₀)
현재 값의 밑이 10인 로그값을 즉시 계산합니다.
10^y = x 를 만족하는 y를 구합니다. 예) 1000 log → 3  |  100 log → 2
ln 자연로그 (logₑ)
현재 값의 밑이 자연상수 e인 로그값을 즉시 계산합니다.
e^y = x 를 만족하는 y를 구합니다. 예) 1 ln → 0  |  e ln → 1  |  100 ln → 4.605...
e 자연상수 e (Euler's Number)
자연상수 e ≈ 2.71828을 디스플레이에 입력합니다.
자연로그의 밑으로, 지수 성장·감쇠에 사용됩니다. 예) e x^y 2 = → e² ≈ 7.389
